# Jeya Jothi - Portfolio

A modern, responsive developer portfolio showcasing my projects, skills, and experience.  
Designed with a clean UI, smooth interactions, and performance-focused architecture.

## Features

- Responsive layout (mobile-first)
- Project showcase with live & GitHub links
- Glassmorphism gallery modal with image slider
- Auto-looping project galleries using Swiper
- Clean UI with subtle animations
- Accessible & keyboard-friendly interactions

## Tech Stack

- React
- Tailwind CSS
- JavaScript (ES6+)
- Swiper.js
- React Icons

## Projects

Each project includes:

- Description & tech stack
- Live preview & source code
- Image gallery popup with smooth transitions

## Getting Started

```bash
git clone https://github.com/Jeya-Jothi/Portfolio_Jo.git
cd Portfolio_Jo
npm install
npm run dev
```
